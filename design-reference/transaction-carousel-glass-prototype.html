Transaction Carousel Concept

This keeps your existing home-rebuild glass style and turns transaction overview into a Wallet-like card swipe surface. The first touch target is the carousel, then users drop into line-item details below.

Glass Surface Scroll Snap Carousel Wallet Card Mental Model

Interaction Model

Horizontal scroll with snap. Active card scales up. Pagination dots support direct jumps. This mirrors the "swipe card, inspect segment" behavior from wallet patterns.

Why It Fits Home Rebuild

Same hero gradient, same shell and tabbar language, same SF typography. We only changed content hierarchy, not your visual DNA.

Next Iteration Hook

We can bind each card to a filtered transaction feed and animate the list transition when the active card changes.
